🔧 CORS问题修复测试

❌ 原始问题：

错误信息： Access to XMLHttpRequest blocked by CORS policy

原因： 前端(http://192.168.219.130:5173) 访问后端(http://localhost:8101) 被浏览器阻止

✅ 修复方案已应用：

  • ✅ 更新了 vite.config.js 添加代理配置
  • ✅ 修改了 .env 文件使用相对路径
  • ✅ 后端添加了 @CrossOrigin 注解
  • ✅ 创建了全局CORS配置类

🧪 登录功能测试

📋 修复详情

1. Vite代理配置 (vite.config.js):

proxy: {
  '/api': {
    target: 'http://localhost:8080',
    changeOrigin: true,
    secure: false
  }
}

2. 环境变量配置 (.env):

VITE_API_BASE_URL=

3. 后端CORS配置:

@CrossOrigin(origins = {"http://localhost:5173", "http://192.168.219.130:5173"}, allowCredentials = "true")
